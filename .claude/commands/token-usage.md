# 토큰 사용량 표시

현재 대화 세션의 토큰 사용량을 요약해서 보여준다.

## 표시할 정보

1. **입력 토큰**: 사용자 메시지 + 시스템 컨텍스트로 사용된 토큰
2. **출력 토큰**: Claude 응답으로 생성된 토큰
3. **총 토큰**: 입력 + 출력 합계
4. **예상 비용**: 현재 모델 기준 예상 비용 (USD)

## 출력 형식

```
## 토큰 사용량 현황

| 항목 | 토큰 수 |
|------|---------|
| 입력 토큰 | X,XXX |
| 출력 토큰 | X,XXX |
| 캐시 읽기 | X,XXX |
| 캐시 생성 | X,XXX |
| **총 토큰** | **X,XXX** |

### 예상 비용
- 입력: $X.XX (@ $X.XX / 1M tokens)
- 출력: $X.XX (@ $X.XX / 1M tokens)
- **총 비용**: $X.XX
```

## 참고

- `/cost` 명령어로도 비용을 확인할 수 있습니다
- 토큰 사용량은 대화가 진행됨에 따라 누적됩니다

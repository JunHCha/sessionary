# Folder

여러 Lecture의 Session들을 사용자가 원하는 대로 모아 구성하는 커스텀 재생목록입니다.

## 규칙

- Experimental Plan 이상 사용자만 이용 가능
- 여러 Lecture에서 원하는 Session을 선택하여 구성
- 사용자별로 여러 개의 Folder 생성 가능

## 사용 예시

- 특정 테크닉만 모은 연습 목록
- 좋아하는 곡들의 하이라이트 세션 모음
- 난이도별 연습 커리큘럼 구성
